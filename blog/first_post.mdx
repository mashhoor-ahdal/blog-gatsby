Representations of Numbers in Python:

1. **Decimal**: The default representation without any prefix.
   Example: `decimal_number = 42`

2. **Binary**: Prefix the number with `0b`.
   Example: `binary_number = 0b1010`

3. **Octal**: Prefix the number with `0o`.
   Example: `octal_number = 0o52`

4. **Hexadecimal**: Prefix the number with `0x`.
   Example: `hexadecimal_number = 0x2A`
